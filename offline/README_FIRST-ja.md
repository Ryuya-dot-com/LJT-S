# LJT-S オフライン版（2026-07-10）

このパッケージは、インターネット接続に依存せずLJT-Sを実施するための専用版です。
公開中のGitHub Pages版とは別に配布され、公開URLの動作には影響しません。

## 実施前の準備

1. Zipファイルを展開してください。Zip内から直接 `START_HERE.html` を開かないでください。
2. `START_HERE.html` と `audio` フォルダを同じ場所に置いたまま使用してください。
3. イヤホンまたはヘッドフォンを接続してください。
4. ChromeまたはEdgeの最新版での実施を推奨します。

## 受験方法

1. `START_HERE.html` をダブルクリックします。
2. 匿名IDなど、研究担当者から指定された情報を入力します。
3. 説明と参加前の確認を読み、音声チェックへ進みます。
4. 練習4問と本試験40問を完了します。
5. 終了画面で必ずCSVを保存します。
6. 保存したCSVを研究担当者へ提出します。

## 重要事項

- このオフライン版ではメール送信を無効にしています。結果は自動送信されません。
- CSVが参加者側の正式な結果ファイルです。ブラウザを閉じる前に必ず保存してください。
- `audio` フォルダ内には本試験40本・練習4本、合計44本の音声があります。移動、改名、削除をしないでください。
- 未完了の回答と完了結果は、再開・再ダウンロード用として最大24時間ブラウザに一時保存される場合があります。終了画面の「この端末の保存データを削除」を使用し、共有端末では研究担当者のデータ管理手順に従ってください。
- 氏名を入力した場合、CSVには個人情報が含まれます。承認済みの研究計画に従って保管してください。

## トラブル時

- 音声が再生されない場合は、Zipを展開済みか、`audio` フォルダが `START_HERE.html` と同じフォルダ内にあるかを確認してください。
- ブラウザの音量、OSの出力先、イヤホンの接続も確認してください。
- 途中でページを再読み込みしたり、`START_HERE.html` だけを別の場所へ移動したりしないでください。
