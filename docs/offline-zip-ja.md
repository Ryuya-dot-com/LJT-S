# LJT-S オフラインZip実施手順

## 対象

研究者・大学院生が、ネットワークに依存せずデスクトップ環境でLJT-Sを実施するための手順です。

## 実施前

1. 配布Zipを展開します。
2. 参加者用には `participant.html` を使います。
3. イヤホンまたはヘッドフォンを用意します。
4. 可能であればChromeまたはEdgeで開きます。

## 参加者の手順

1. `participant.html` をダブルクリックします。
2. 匿名IDを確認し、必要に応じて学校・クラスコードを入力します。
3. 説明を確認して音声チェックへ進みます。
4. 練習4問、本試験40問を完了します。
5. 終了画面でCSVを保存します。
6. 研究者の指示に従ってCSVを提出します。

## 研究者の回収

- CSVファイル名には匿名ID、条件、日時が含まれます。
- 実名を入力した場合、CSV/JSONには個人情報が含まれます。承認済みプロトコルに従って保管してください。
- 公開参加者モードのCSVは正答キー・語彙・IRTパラメータを含みません。
- 研究者モードURLから実施したCSVは分析用メタデータを含みます。

## 注意

- オフライン実施ではGoogle Sheetへの自動保存は保証されません。
- ネットワークがあり、`data/submission.js` にGAS URLが設定されている場合は自動送信も試行されます。
- CSVは必ず回収してください。
